# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅ |

## Reporting a Vulnerability

Please report security vulnerabilities **privately** via [GitHub Security Advisories](https://github.com/wenisch-tech/proxera/security/advisories/new).

Do **not** open a public issue for security vulnerabilities.

We aim to acknowledge reports within 48 hours and provide a fix within 14 days for critical issues.
